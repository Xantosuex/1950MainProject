IT Sensations Home

Introdiction to the Protosite

This project is a COMP1950 website redesign. Though there are some good features of the existing site, it is failing in its primary focus: To present information to students in a way that facilitates learning. The way that lessons are organized (chronologically) is good, but it is lacking in additional material, such as code repositories, examples and exercises.

Though the core audience is existing students, it would be nice to acknowledge prospective students and alumni. The new site should better help keep students up-to-date on breaking news and it should be easy to maintain.
