Creative Brief

Project Goals

  • Create a repository of information for students that:
    • Is a comprehensive detail of all course material. This should include examples and exercises for each lesson.
    • Covers breaking news and be easy to maintain.
    • Should contain a wiki/glossary.
    • Acknowledges prospective students and alumni.
  • The site overall should be aesthetically pleasing and should reflect the techniques, concepts and practices of COMP1950 course material.
